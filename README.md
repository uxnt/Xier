# Xier 编程语言( Xie Red Neon Hyper Divine Blade )

```css
标题:系统流打印信息
导入:系统流
导入:工具包
值:整数名字:(5)

函数:主程序:(){
    值:整数名字:(5)
    打印换行:("你好, 世界 !")
    打印换行:("%d", 整数名字)
}

函数:添加名字:(字符串:甲名字) {
    名字:(名字 + 甲名字)
}

类:甲的电脑:(){
    值:电脑名字:"甲的电脑"
    函数:开机:() {
        打印换行:(电脑名字 + "正在启动了...")
    }
}

函数:删除名字:(字符串:名字)

类:机械:(){
    值:名字:"机械"
    名字:() {}
    启动:() {}
    关机:() { 删除名字:(名字) }
    停机:() {}
}

函数:删除名字:(字符串:名字) {
    释放:名字
}


字符串:获取机械名字:() {
    机械:分析器:"分析器"
    // 字符串:名字:("机械名字: " + 分析器:名字)
    返回:("机械名字: " + 分析器:名字)
}



```
